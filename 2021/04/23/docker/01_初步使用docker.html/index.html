01_初步使用docker

摘要
第一步:安装 Docker
终端执行curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
注意:在我的 debian 的使用中,每次都需要sudo
为了不节省可以使用以下办法解决
sudo usermod -aG docker ${USER}
sudo service docker restart
重启操作系统即可
